# Figma: кликабельный flow (готовая карта переходов)

## Фреймы (имена в Figma)

1. `06 Вход`
2. `07 Выбор роли и станции`
3. `01 Обзор дашборд`
4. `02 Сортировка`
5. `03 Сканирование Washing QC`
6. `04 Выдача Pickup`
7. `05 Детали заказа`
8. `08 Нет доступа`

## Базовая логика

1. Любой пользователь начинает с `06 Вход`.
2. После входа попадает в `07 Выбор роли и станции`.
3. Из `07` переходит только в разрешенные ему станции.
4. Попытка открыть запрещенную станцию ведет в `08 Нет доступа`.

## Prototype Connections (что соединить)

### 06 Вход

1. Кнопка `Войти` -> `07 Выбор роли и станции`
   - Trigger: `On click`
   - Action: `Navigate to`
   - Animation: `Smart animate`, `300ms`

### 07 Выбор роли и станции

1. Кнопка `Открыть` в карточке `Sorting` -> `02 Сортировка`
2. Кнопка `Открыть` в карточке `Washing / QC` -> `03 Сканирование Washing QC`
3. Кнопка `Открыть` в карточке `Pickup` -> `04 Выдача Pickup`
4. Кнопка `Открыть` в карточке `Overview / Детали` -> `01 Обзор дашборд`

Дополнительно для демонстрации RBAC:
5. На любой "чужой" карточке сделай отдельный hotspot `Нет доступа` -> `08 Нет доступа`

### 01 Обзор дашборд

1. Клик по карточке заказа в колонке `Не отсортировано` -> `02 Сортировка`
2. Клик по карточке в колонке `Washing` -> `03 Сканирование Washing QC`
3. Клик по карточке в колонке `Готово к выдаче` -> `04 Выдача Pickup`
4. Клик по KPI/карточке заказа -> `05 Детали заказа`

### 02 Сортировка

1. Кнопка `Создать корзины + Печать QR` -> `03 Сканирование Washing QC`
2. Кнопка `Назад` (добавь в Figma) -> `01 Обзор дашборд`

### 03 Сканирование Washing QC

1. Кнопка/область `Статус обновлён успешно` -> остается на этом же экране (`Open overlay` или `Navigate to self`)
2. Кнопка `Открыть журнал` -> `05 Детали заказа`
3. Кнопка `Перейти к выдаче` (добавь) -> `04 Выдача Pickup`

### 04 Выдача Pickup

1. Кнопка `Выдано клиенту` -> `01 Обзор дашборд`
2. Кнопка `Открыть` у заказа -> `05 Детали заказа`

### 05 Детали заказа

1. Кнопка `Назад к обзору` (добавь) -> `01 Обзор дашборд`
2. Кнопка `К выдаче` (добавь) -> `04 Выдача Pickup`

### 08 Нет доступа

1. Кнопка `К выбору` -> `07 Выбор роли и станции`
2. Кнопка `Выход` -> `06 Вход`

## Демо-сценарий (для показа)

1. `06 Вход` -> `07 Выбор роли`.
2. Открыть `Sorting` -> показать создание корзин.
3. Перейти в `Washing/QC` -> показать scan-first.
4. Перейти в `Pickup` -> завершить выдачу.
5. Открыть `Детали заказа` -> показать аудит.
6. Отдельно показать попытку открыть чужую станцию -> `Нет доступа`.

## Режим презентации

1. Включить `Present`.
2. Поставить device frame: `Desktop 1440`.
3. Для каждого экрана в заметках добавить:
   - роль,
   - цель экрана,
   - критический action.
