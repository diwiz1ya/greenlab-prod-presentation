# MVP-прототип для показа тимлиду (RU)

В папке лежат 5 экранов в формате `SVG`, подготовленные для быстрого импорта в Figma:

1. `01_Обзор_дашборд.svg`
2. `02_Сортировка.svg`
3. `03_Сканирование_Washing_QC.svg`
4. `04_Выдача_Pickup.svg`
5. `05_Детали_заказа.svg`

Дополнительно для UX-логики и доступа:

6. `06_Вход.svg`
7. `07_Выбор_роли_и_станции.svg`
8. `08_Нет_доступа.svg`

## Как открыть в Figma

1. Открой Figma и создай новый файл.
2. Перетащи все 5 `SVG` прямо на холст.
3. Выдели все экраны и выровняй по горизонтали.
4. Переименуй страницы/фреймы при желании:
   - Обзор
   - Сортировка
   - Сканирование
   - Выдача
   - Детали заказа

## Что говорить на демонстрации

Используй файл `Сценарий_демо.md` как готовый текст для объяснения.

Для кликабельного прототипа открой:

- `Figma_Кликабельный_Flow_связи.md` (полная карта переходов)

## Примечание

Это MVP-макет под задачу "показать готовый дашборд", с акцентом на ключевой поток:
`Новый заказ -> Сортировка -> Сканирование этапов -> Выдача -> Завершение`.
